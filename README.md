# Akıllı Kütüphane Yönetim Sistemi
Bu proje, Java programlama dili kullanılarak geliştirilmiş bir kütüphane otomasyon sistemidir.
Uygulama, temel kütüphane işlemlerini gerçekleştirmek amacıyla tasarlanmış olup, mimarisinde Nesneye Dayalı Programlama (OOP) yapıları kullanılmıştır.

**Temel Özellikler:**
* Kitap ekleme ve listeleme.
* Öğrenci ekleme ve listeleme.
* Kitap ödünç verme ve teslim alma.
* Kitap ödünç verme listesini görüntüleme.
